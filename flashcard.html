JavaScript

JavaScript is a programming language used to make web pages interactive.

Operators

Assignment operator: Changes the value of the variable on the left of the operator.

Comparison operator: Does not change any values, but returns a boolean (true or false) depending on whether the statement evaluates as true or false.

Variables

Variables can be thought of as containers used to store information. They allow for a way to label data with a descriptive name.

Variable Naming Convention

Lower camel case: Use lowerCamelCase when naming JavaScript variables. Start with a lowercase letter, and if the variable name is more than one word, remove all spaces and capitalize the first letter of each subsequent word.

Functions

Function: A function is a block of code that performs an action and returns a result; optionally takes arguments

Methods

Method: A method is an action run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response.

Arguments

An argument is what you pass into a function or method

Parameters

A parameter is a variable that's assigned to the argument

Return

Once called, many functions will provide a result of some kind. This is known as returning, and the result is often referred to as a return value.

Chaining Methods

Chaining methods is calling a method directly on the return value of another method.

Strings

String: a JavaScript data type that represents the exact text of whatever is enclosed in the quotes.

Boolean

Boolean: true or false

Undefined

Undefined: An object (such as a variable) without a defined value.

NaN

NaN:NaN stands for not a number yet is a type of number.

Escape

The \\ around either side of the text above is called an escape. It tells JavaScript that the " that comes right after it is not the end of the string, but just a character inside the string.

Alert

alert() opens a dialog box and returns undefined.

Comments

Comment: anything following // on a line in JavaScript is ignored by the computer. Comments are a convenient way to leave notes in your code for yourself or other programmers.

jQuery

jQuery: a JavaScript library that makes it easy to make web pages interactive.

Attributes

In computing, an attribute is a specification that defines a property of an object, element, or file.
